Interactive Ambient Sound Installation

This project is an exploration of methods of interacting with music via non-traditional methods
The use of different interfaces is intended to allow for creative expression sonically without any musical training

Using different sensors such as a gyroscope and pressure-sensitive strips into arduinos and wifi transmitters
I am able to wirelessly transmit data of physical interactions to a computer.
Here, the data can change parameters of ambient sounscape I created in MaxMSP

Still a work in progress,amd I hope to incorporate interactive visuals.
